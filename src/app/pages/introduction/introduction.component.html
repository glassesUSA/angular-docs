Getting Started

What is a design system?

When creating content, designing, photographing and so on, a uniform identity must be maintained for all final products. To make this task easier, this Design System includes rules, explanations, assets and everything needed to easily create a large and uniform environment.

How to use?

To use the design system head to the Getting Started section and follow the instructions there.

Components
