Usage
To install the styling and scripts, include the following code in the section of your project. <!-- CSS Styling -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/glassesUSA/dsm@latest/dist/styles.css">

<!-- Scripts - Required if you are using any form elements/buttons-->
<script type="module" src="https://cdn.jsdelivr.net/gh/glassesUSA/dsm@latest/dist/dsm-effects.js"></script> Additionally, in order to use the widgets with Swiper functionality such as the trustpilot widgets, include this script tag. <script src="https://cdnjs.cloudflare.com/ajax/libs/Swiper/6.4.14/swiper-bundle.min.js" integrity="sha512-APQL/QFRDoEBh9V5ztds1P6qEICllWq9Dpcv+pVu5DE/g5MZ1GE7+pj9tHhN1SFBv4J7B2j5mfw0cOUitz+aGg==" crossorigin="anonymous"></script> If you cannot include scripts in the head tag, use the following script to include the files into your document. 
<script>
    var insertScript = document.createElement("script");
    insertScript.src = 'https://cdn.jsdelivr.net/gh/glassesUSA/dsm@latest/dist/dsm-effects.js'
    insertScript.type = 'module'
    document.head.appendChild(insertScript)
</script>
